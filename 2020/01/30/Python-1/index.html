Python 1

​ It is the first time when I learn the python. Python is a high-level general purpose programming language. In fact it is not easy. Luckily ,I have learned VB for one semester before, which share some similarities with python, so it would be easier for me to learn about python. Now I would like to introduce a code segment which I learned recently.

​ It is an example of “continue” statement which is used in the loop, if the process runs into “continue” ,it will jump to the beginning of the loop and revalue the loop conditions.

​ This code segment means when you see ‘Who are you?’ and you type ‘Joe’ ,you will be allowed to type your password; otherwise you will be ordered to input the name repeatedly. After you type the right name you will see another empty box if you type ‘swordfish’ you will see ‘Access granted’; otherwise you will back to the beginning of the loop where you will see the question ‘who are you?’

​ After we understood this code segment, our teacher raised a further question, “How to improve the code segment to let people only have 3 chances to enter the password? “

According to his question I changed the code segment as the following section.

​ I added a new variable quantity “n” to work as a counter. If people input wrong passwords three time, “break” will force the process stop, so they can not input the anything. This code segment can work as normal operation for login. However , when I wrote this code I met a question. At first, I did not add a “break” in the first loop which would lead to a consequence that if people do not input the right password, they will type the name again. This do not match our habits. As a result , when I realized this question I decided to use two loops in order to prevent this from happening. When people type the right information, the first loop will finish, not influenced by the second loop.
